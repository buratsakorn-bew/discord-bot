# Bot Discord
## วิธีเปิดใช้งาน
### สิ่งที่ต้องมี 
* NodeJS version > 12
* Discord bot token [Link](https://discord.com/developers/applications)
* Youtube Data API V3 

### ขั้นตอนการลง
* clone/download repository 
* $ npm install 
* ตั้งค่า Token, Prefix, Bot token, Youtube Api ในไฟล์ config.json 

### วิธีใช้งาน
#### Run file runbot.bat
> หรือ
#### $ npm start

### หรือต้องการเชิญ Bot เข้า Discord
