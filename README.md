# Bot Discord
## Require
* NodeJS version 12^
* Discord bot token [Link](https://discord.com/developers/applications)
* Youtube Data API V3 

## Install
* clone/download repository 
* npm install 
* setting Token, Prefix, Bot token, Youtube Api In config.json 

### HowTo Use

```bash
$ npm start
```
